Temperature Prediction Model

Overview

A LightGBM regression model that predicts next-day temperature using today's atmospheric conditions. Trained on over 64 million NOAA Global Surface Summary of Day (GSOD) records.

R²0.9669
MAE2.88°F
Within ±5°F~92%
Training Rows64M

Key Features Used

TEMPDEWPMAX/MINWDSP SLPPRCP7-day lag14-day lag Rolling meansCyclical month/day

How It Works

The model uses gradient boosted decision trees with 45 engineered features including lag temperatures (1, 3, 7, 14 days), 7 and 30-day rolling averages, cyclical sine/cosine encodings for seasonality, and atmospheric pressure trends.

Rain Prediction Model

Overview

A binary classification model that predicts whether rain will occur tomorrow. Also outputs a probability score (0–100%) indicating confidence in rain occurring.

TaskBinary
OutputYes / No
Probability0–100%
AlgorithmLightGBM

Key Features Used

PRCP lagDEWPSLP trendWDSP Humidity proxyRolling precipSeason

How It Works

Uses dew point depression, precipitation history, and pressure trends as primary signals. Low pressure combined with high dew point and recent precipitation are the strongest rain predictors. The model outputs both a hard label and a probability via predict_proba.

Extreme Weather Alert Model

Overview

A multi-class classification model that identifies extreme weather conditions. Outputs one of 5 categories to help users prepare for dangerous weather events.

Class 0✅ Normal
Class 1🔥 Heat
Class 2🥶 Cold
Class 3⛈️ Storm
Class 4❄️ Snow

Key Features Used

TEMP extremesWDSPPRCP SNDPSLP dropsGUSTSeason

How It Works

Detects threshold-crossing patterns: temperatures above 95°F trigger heat alerts, below 10°F trigger cold alerts. Rapid pressure drops combined with strong winds signal storm risk. Snow depth and sub-freezing temps activate snow advisories.
